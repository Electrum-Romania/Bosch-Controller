◆ keyboard_steer_control

std::uint8_t Options::keyboard_steer_control

Manual steer control.

If it is 1, the motors will be controlled by the keyboard input. If it is 0, the motors will be controlled automatically (the car will be in self-driving mode).

Note
Default value is 1.
Used by: Motors.
